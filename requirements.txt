# ========================================
#
# 本项目采用以下技术栈:
#
# 📱 Claude Desktop (MCP 客户端)
#     ↕️ MCP Protocol
# 🔌 EMQX MCP Server (本文件依赖)
#     ↕️ MQTT over SSL/TLS  
# ☁️ EMQX Cloud (托管 MQTT Broker)
#     ↕️ MQTT Topics
# ⚡ eKuiper (Docker 容器)
#     ↕️ REST API
# 🏭 EdgeX Foundry (Docker 容器)
#     ↕️ Virtual Device Protocol
# 🌡️ 虚拟温湿度传感器
#
# Docker 组件无需 Python 依赖，通过 REST API 交互。
# 本 requirements.txt 主要用于 MCP Server 和监控脚本。
